## About this simple 'Games' Application

### Installation instructions

- (optional) Install Homebrew
- Install composer for dependencies
- Install the shell script -- run **./init.sh OR bash init.sh**

If you've got issues with the installation script run this command in the terminal: **run php artisan migrate && php artisan db:seed && php artisan storage:link**
